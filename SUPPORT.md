## Before opening an issue

1. Search existing issues and the Wiki FAQ
2. Run `cargo test --workspace` (or `./scripts/test_all.sh`)
3. Include OS, Rust version (`rustc -V`), and exact commands / logs

Maintainers may close issues that are questions already answered in the Wiki FAQ without new information.
